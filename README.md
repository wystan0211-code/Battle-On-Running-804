# 八愛逃走中！Battle On Running

## 部署到 GitHub Pages

1. 建立新的 Public repository
2. 上傳所有 5 個檔案（index.html, manifest.json, sw.js, icon-192.png, icon-512.png）
3. Settings → Pages → Branch: main → Save
4. 等約 1 分鐘後獲得網址

## 主辦人帳號
- 帳號：`host`  密碼：`804onrunning`

## 玩家加入主畫面
- iPhone：Safari → 分享 → 加入主畫面
- Android：Chrome → 安裝應用程式
